This page lists the software and hardware requirements for running the CloudSim-HO-Research-V2 project.

## Software Requirements

- **Java Development Kit (JDK) 21:** The project is built using Java 21.
- **Maven 3.9+:** Required for building the project and managing dependencies.
- **Git:** For cloning the repository.

## Hardware Requirements

- **64GB of RAM:** Recommended for running the full experimental suite.
- **30GB of free disk space:** For the source code, build artifacts, and log files.
